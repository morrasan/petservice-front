# Pet Service - front
The simple Pet service for the store pets

## Stack version
1. Vue 3.4.21
2. Node 19.5.0
3. Vuetify 3.5.16

## Install / run
1. git clone https://github.com/morrasan/petservice-front.git
2. docker compose up
3. open browser http://localhost:3003

## Environment
1. port 3003
2. http://localhost:8008 - API_URL
